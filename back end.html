    • Menjadi Back End Developer untuk Pemula

    Back end merupakan bagian dari situs web yang tidak dilihat oleh pelanggan. Back end berurusan dengan data-data di balik layar, ia bertanggung jawab dalam menyimpan dan mengatur data, serta memastikan hal yang ada pada sisi klien berfungsi dan tidak bermasalah. Back end bertujuan untuk mengelola data yang digunakan oleh situs web atau aplikasi. Ini termasuk menyimpan data pengguna, produk, pesanan, dan semua informasi lain yang diperlukan oleh platform.

    Kurikulum

    1. Pengenalan apa itu Back End Developer
    2. Belajar Node.js
    3. Belajar Node.js NPM
    4. Belajar Express.js
    5. MySQL Database
    6. PostgreSQL Database>
    7. Belajar RESTful API
    8. Belajar Testing Api menggunakan postman untuk pemula
